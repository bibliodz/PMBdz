O arquivo deve estar no formato UNIMARC<br />
<ul>
<li>A importação é feita em 3 passos:
<OL>
<LI>Upload do arquivo</LI>
<LI>Preload do arquivo</LI>
<LI>Transferir para o registro do banco de dados</LI>
</OL></li>
<br /><li>Ao invés de fazer upload do arquivo, é possível enviá-lo diretamente para o servidor por FTP, por exemplo. O arquivo deve ser nomeado <i>"!!nom_fic!!"</i> e alocado em admin/import. Neste caso, você deve ir diretamente para o passo <b>"!!nom_bouton!!"</b>.</li>
</ul>
